# Nuxt Full-Stack Starter Template

Template profesional para proyectos SaaS con Nuxt 4

## Stack Tecnológico

- **Framework:** Nuxt 4
- **Base de Datos:** PostgreSQL
- **ORM:** Prisma
- **Autenticación:** Auth.js (NextAuth)
- **Estilos:** TailwindCSS
- **Validación:** Zod
- **Utilidades:** VueUse, Dayjs, Nanoid, Bcrypt
